## Data Types

### Numerical - Basic

- `INT` for integers
- `BIGINT` for bigger integers
- `FLOAT` for floating points numbers.
- `DOUBLE` for bigger floating point numbers.

### Strings - Basic

- `CHAR` - fixed length string
- `VARCHAR` -  variable length string
